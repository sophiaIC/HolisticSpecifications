For the introduction I wanted to start with two good agents. If this part is okay I will try to extend it so that it contains agents of unknown providence (hence needing an honest third party).  
\\\\
\textbf{Spec} Wallet

    \textbf{field} isEuro : Bool  ---- false means the wallet contains dollars
    
    \textbf{field} value : Nat
\\\\    
\textbf{Spec} EWallet : Wallet
   
    \textbf{inv} isEuro = true
    \\\\
\textbf{Spec} DWallet : Wallet
  
    \textbf{inv} isEuro = false
    \\\\
\textbf{Spec} Agent

    \textbf{field} ew : EWallet
    
    \textbf{field} dw : DWallet
\\\\
\textbf{Spec} CanTrade  (a1,a2,ae : Agent, num, rate : Nat) : Bool

---- ae is the agent who wants to trade euros

---- rate is the exchange rate from euros to dollars

---- num is the number of euros that are to be exchanged

\textbf{return}

(a1 $\neq$ a2)  $\wedge$  (ae.value $\geq$ num) $\wedge$

((ae = a1) $\wedge$ (rate * num $\leq$ a2.dw.value))  $\vee$

((ae = a2) $\wedge$ (rate * num $\leq$ a1.dw.value)))
